How to open and run eGrocer Website?

  1. Open your react project in VS Code or any code editior you have.

  2. Open terminal and change your current directory path to the folder which contains package.json file of your react project.


  3. Now insall all the dependencies. For that run a command "npm install" in your terminal.


  4. Make a build folder of your project. For that run a command "npn run build"


  5. It will create build folder. Now run a command "serve -s build" to your terminal to run your project. It will serve your project and provides you local and network address. Copy one of those link and open in any browser. Then you can see your project running successfully
