How to open and run eGrocer Website?

  1. Open your react project in VS Code or any code editior you have.

  2. Open terminal and change your current directory path to the folder which contains package.json file of your react project.

  3. Now install all the dependencies. For that run a command npm install in your terminal.

  4. To run the website in your localhost, Run this command npm start

  5. Make a build folder of your project. For that run a command npm run build


How to deploy website on server?

  1. There are two ways to deploy website on server
    1. Deploy by build zip folder
      • Create zip file of the build folder from the website code
      • Upload it on website root folder and extract thair content in the root folder
    2. Deploy ftp
      • To deploy website via FTP you have to create ftp of the website
      • Set ftp details in the ftp.js file

      • After setup ftp properly, run this code to deploy website on server npm run deploy
