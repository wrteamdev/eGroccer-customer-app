Server Requirement

For managing dependencies, Laravel uses composer. Make sure you have a Composer installed on your system before you install Laravel.

  1. PHP >= 8.1
  2. BCMath PHP Extension
  3. Ctype PHP Extension
  4. Fileinfo PHP extension
  5. JSON PHP Extension
  6. Mbstring PHP Extension
  7. OpenSSL PHP Extension
  8. PDO PHP Extension
  9. Tokenizer PHP Extension
  10. XML PHP Extension

Configure PHP INI basic settings

For managing dependencies, Laravel uses composer. Make sure you have a Composer installed on your system before you install Laravel.

  1. max_execution_time : 3000
  2. memory_limit : 500M
  3. post_max_size : 200M
  4. upload_max_filesize: 200M

Installation Steps for Setup in Localhost

  1. Install XAMPP and Composer.
  2. Copy egrocer code in (EX. C:/Xampp/htdocs/egrocer) your folder (make sure all files move). If you going to setup it on server give root path to public folder (public_html/egrocer/public).
  3. Add Your Domain Name As APP_URL=yourdomain (Ex: wrteam.in or egrocer.wrteam.in) in .env File. (Note:Only for Live Server)
    4. Note: Be advised that if you do not find the .env file in the root folder, it is necessary to activate the "Unhide hidden files and folders" option in your settings.
  4. Create Database name 'egrocer' and change DB_DATABASE=egrocer DB_USERNAME=database_user_name DB_PASSWORD=database_user_password in .env File.
  5. Now, in root directory, run this command composer install
  6. After Composer installed successfully, run this command npm install
  7. After npm installed successfully, run this command php artisan migrate to add all required tables in database. (Note: Make Sure Database configure in .env)
  8. After Migration run this command to add required database entries php artisan db:seed
  9. For API's Authentication run command php artisan passport:install
  10. For Images run command php artisan storage:link, If it is show link is already generated Please delete folder public/storage and run command again.
  11. Now, in root directory, run this command php artisan serve (Note: This is only for localhost not for server)

Installation Steps for Setup in Server

  1. Upload admin panel code eGrocer Admin Panel.zipzip file on your server admin panel root folder
  2. Extract that file's content in that admin panel's folder, Make sure all content should be in admin panel's root folder, Includes .editorconfig, .env, .env.example, .htaccess, .styleci.yml files
    3. Note: Be advised that if you do not find the .env file in the root folder, it is necessary to activate the "Unhide hidden files and folders" option in your settings.
  3. Open your admin panel url and add postfix /install (Like: https://admin_panel_url.com/install)



  4. These are the minimum requirements, These all should be green marked, If you get any red mark here, Enable that plugin from the server or Ask server support to enable that plugin and make sure all will green marked



  5. Keep Database host and Database port value same, Set Database name, Dabase username, Database password will be set here your created database details and email and password as per your requirement (This email and password will use for login the admin panel)



  6. Enter purchase code here provided by codecanyon (How to get purchase code?)



  7. Wohoo! Your admin panel setup done, Congratulations!



  8. Now open your admin url, You will get working admin panel there.
